# Mindly Dock
Dumb Phone method for iOS written in C++, leveraging libimobiledevice and iOS profiles

## Build

### Dependencies (macOS)
```bash
brew install glfw libimobiledevice libplist
```

### Compile
```bash
mkdir build && cd build
cmake ..
make
```
